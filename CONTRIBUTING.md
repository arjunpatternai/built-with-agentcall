# Contributing to built-with-agentcall

This repo is a showcase of real things people build on **[AgentCall](https://agentcall.dev)** — each one
a self-contained folder. Built something? Add it here.

## Add your project

1. **Fork** this repo.
2. **Create a folder** at the repo root for your project — a short, descriptive, kebab-case name (e.g. `meeting-notetaker/`).
3. Put your code in it, with its **own `README.md`** (what it does + how to run it). The [`meeting-notetaker/`](meeting-notetaker) folder is a good shape to copy.
4. **Don't commit secrets** — no API keys, no `.env` files. Add a `.gitignore`.
5. Open a **Pull Request**. We review and merge it; your commits stay credited to you.

## Folder conventions

- One project = one top-level folder, runnable on its own.
- Each folder has its own `README.md`.
- Keep it self-contained — no cross-folder dependencies.
- By contributing, you agree your folder ships under this repo's license (MIT).

## What makes a good entry

- It actually runs — include setup steps in your README.
- It clearly uses AgentCall.
- A short GIF or screenshot in your README goes a long way.

Questions? Open an issue. Thanks for building.
